Stars: 3 · Languages: Rust · **Active**

### [saige](https://github.com/urmzd/saige)

Stars: 2 · Languages: Go, Go Template · **Active**

### [teasr](https://github.com/urmzd/teasr)
Single-binary Rust tool for capturing showcase screenshots and GIFs from web apps, desktop, and terminal, with zero runtime dependencies. Streamlines visual documentation for developers and teams.
Stars: 2 · Languages: Rust, HTML · **Active**

### [experimental-transformer-architectures](https://github.com/urmzd/experimental-transformer-architectures)
Seventeen experimental language-model architectures where the hidden dimension equals vocabulary size—eliminating embeddings and output projections so every intermediate state is a vocabulary distribution. Explores unconventional transformer designs in Python.
Languages: Python · **Active**
