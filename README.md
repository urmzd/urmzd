## Spotlight

### [incipit](https://github.com/urmzd/incipit)
Template-driven CLI that transforms structured resume data into polished PDF, DOCX, HTML, LaTeX, and Markdown, with pluggable templates and AI-powered resume review and optimization. Built in Go, it streamlines multi-format resume generation and integrates ready-to-pipe AI prompts.
Stars: 12 · Languages: Go, HTML, TeX · **Active**

### [mnemonist](https://github.com/urmzd/mnemonist)
Local-first Rust CLI for AI agent memory, supporting cognitive verbs, plain-markdown memories, local candle embeddings, and HNSW semantic search across memory and code. Benchmarked on LongMemEval, it enables tool-agnostic, semantic memory management for agents.
Stars: 5 · Languages: Rust, Python · **Active**

### [saige](https://github.com/urmzd/saige)
Unified Go SDK, CLI, and MCP server for streaming AI agents, knowledge graphs, and RAG pipelines, integrating providers like Ollama, OpenAI, Anthropic, and Google behind a single interface. Designed for seamless orchestration of advanced AI workflows.
Stars: 4 · Languages: Go, Go Template · **Active**

### [agentspec](https://github.com/urmzd/agentspec)
Universal agent skill and sub-agent manager with a terminal user interface, written in Rust and Shell. Enables modular management and orchestration of agent capabilities through a TUI.
Stars: 2 · Languages: Rust, Shell · **Active**

### [linear-gp](https://github.com/urmzd/linear-gp)
Production-grade Rust framework for Linear Genetic Programming research, featuring modular architecture, Q-Learning integration, and automated hyperparameter optimization. Supports reinforcement learning and classification tasks in a research-ready environment.
Stars: 2 · Languages: Rust · **Active**
